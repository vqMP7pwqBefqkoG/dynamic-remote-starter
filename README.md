# Dynamic Remote Starter

Dynamic Remote Starterは、Webブラウザを通じてWindowsのバッチファイル（.bat）をリモートで起動、停止、管理するためのシンプルなアプリケーションです。PCの前にいなくても、スマートフォンなどから手軽にアプリケーションを操作できます。

## 機能

*   **アプリケーションの追加・削除:** バッチファイルへのパスとポート番号を登録して、アプリケーションを管理します。
*   **アプリケーションの起動・停止:** 登録されたアプリケーションをWeb UIから簡単に起動・停止できます。
*   **ステータス表示:** 各アプリケーションの現在の実行状態（Run/Stop）をリアルタイムで確認できます。
*   **カードの並べ替え:** ドラッグ＆ドロップでアプリケーションカードの表示順を自由に並べ替え、カスタマイズできます。
*   **ダークモード:** 好みに合わせてライトテーマとダークテーマを切り替えられます。
*   **モバイルフレンドリー:** スマートフォンからも快適に操作できるレスポンシブデザインです。
*   **動的リンク:** 実行中のアプリケーション名をクリックすると、アクセス元のIPアドレスに適応したURLでアプリケーションを開きます。

## セットアップ

### 前提条件

*   Python 3.x がインストールされていること
*   `pip` が利用可能であること

### インストール

1.  このリポジトリをクローンします。
    ```bash
    git clone https://github.com/vqMP7pwqBefqkoG/dynamic-remote-starter
    cd dynamic-remote-starter
    ```
2.  必要なPythonライブラリをインストールします。
    ```bash
    pip install Flask
    ```
3.  **`SECRET_PATH` の設定 (重要！)**
    `app.py` を開き、`SECRET_PATH` の値をユニークな文字列に変更してください。これは、不正なアクセスからアプリケーションを保護するための簡易的なセキュリティ対策です。
    ```python
    # --- 設定 ---
    PORT = 9999
    SECRET_PATH = "ここにユニークな文字列を設定してください" # 例: "my-secret-admin-panel-12345"
    ```

### サーバーの起動

`start_server.bat` ファイルを実行してサーバーを起動します。

サーバーが起動すると、コマンドプロンプトにアクセスURLが表示されます。

## 使用方法

### Web UIへのアクセス

ブラウザで以下のURLにアクセスします。

*   **PCから:** `http://127.0.0.1:9999/`
*   **スマートフォンなど、別のデバイスから（VPN接続時）:** `http://<あなたのPCのIPアドレス>:9999/`
    *   PCのIPアドレスは、コマンドプロンプトで `ipconfig` コマンドを実行するか、TailscaleなどのVPNサービスを利用している場合は `tailscale ip` コマンドで確認できます。

### アプリケーションの追加

1.  ページ下部にある「Add New Application」セクションを展開します。
2.  **Application Name:** アプリケーションの分かりやすい名前を入力します（例: `ComfyUI`）。
3.  **Port:** アプリケーションがリッスンするポート番号を入力します（例: `8188`）。リンク機能を利用しない場合は空欄でも構いません。
4.  **Full Path to .bat file:** 起動したいバッチファイル（.bat）のフルパスを入力します。""は必要ありません。
    *   **ヒント:** Windowsでは、Shiftキーを押しながらファイルを右クリックし、「パスのコピー」を選択すると、フルパスを簡単に取得できます。
5.  「Add」ボタンをクリックしてアプリケーションを追加します。

### アプリケーションの管理

*   **起動 (▶):** アプリケーションを起動します。
*   **停止 (■):** 実行中のアプリケーションを停止します。
*   **削除 (🗑️):** アプリケーションをリストから削除します。

### カードの並べ替え

アプリケーションカードをドラッグ＆ドロップすることで、表示順を自由に並べ替えることができます。並べ替えた順序は自動的に保存されます。

### ダークモード

ページ下部のフッターにある🌙/☀️アイコンをクリックすることで、ライトテーマとダークテーマを切り替えられます。設定はブラウザに記憶されます。

### 動的リンク

実行中のアプリケーション名をクリックすると、そのアプリケーションがリッスンしているポート番号で新しいタブが開きます。このリンクは、Web UIにアクセスしているデバイスのIPアドレスに合わせて自動的に調整されます。
